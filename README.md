# Enterprise-Network-Design-CCNA-CCNP
Professional enterprise network infrastructure project (CCNA/CCNP level) including VLAN segmentation, Layer 3 switching, HSRP redundancy, OSPF dynamic routing, NAT (PAT), and advanced security (ACLs, SSH, Port Security). Designed for scalability, high availability, and real-world deployment.
 Project Title

Enterprise Network Design & High Availability Implementation

📝 Description

This project demonstrates a complete enterprise-grade network infrastructure for a medium-sized company (150 users). The design incorporates VLAN segmentation, inter-VLAN routing, HSRP for high availability, OSPF dynamic routing, NAT (PAT), and advanced security mechanisms including ACLs, SSH, and Port Security. All configurations are based on Cisco technologies, suitable for CCNA and CCNP level certification practice and professional deployment.

 Objectives
-Provide reliable and redundant network access for all users
-Ensure high availability using HSRP
-Enable secure remote management via SSH
-Implement VLAN-based segmentation for departments and services
-Configure NAT (PAT) for Internet access
-Apply ACLs for traffic filtering between VLANs


🛠️ Technologies & Tools
-Cisco Packet Tracer 
-Cisco IOS commands and configurations
-VLANs, STP, HSRP, OSPF, NAT (PAT), ACLs, SSH

Features & Highlights
-VLAN Segmentation: Users, Admin, Servers, VoIP, Management, WiFi
-Inter-VLAN Routing: Layer 3 SVI on core switches
-High Availability: HSRP active/standby configuration
-Routing: OSPF dynamic routing between core and edge
-Internet Access: NAT (PAT) configuration on Edge router
-Security: SSH access, ACL standard & extended, port security
-Testing & Troubleshooting: Scenarios for failover, VLAN issues, NAT, SSH, and network loops

Testing & Troubleshooting
-HSRP Failover: Simulate Core-01 failure to test Core-02 takeover.
-VLAN Connectivity: Verify inter-VLAN routing via ping and trace commands.
-NAT / Internet Access: Confirm internal hosts can reach the Internet.
-SSH Security: Ensure console and VTY access is secured.
-Loop Prevention: Validate STP configurations.

Outcome

This project equips network engineers with practical experience in designing, implementing, and securing a medium-sized enterprise network, following best practices for high availability, security, and scalability.
